Automated Testing Project – Pinterest Web Application

## 📌 Project Overview
- This project focuses on automated testing of the Pinterest Web Application using Katalon Studio.
- The goal is to validate core functionalities and ensure system stability through automated test execution.

## ✅ Objectives
- Validate core user flows (login, search, interaction)
- Ensure system behaves correctly under different scenarios
- Improve test coverage using automation
- Enhance testing efficiency and reduce manual effort

## 🛠 Tools Used
- Automation Tool: Katalon Studio
- Testing Type: Black-box testing
- Platform: Web Application

## 📂 Deliverables
- Test Cases: Designed based on system requirements
- Test Scripts: Automated test scripts in Katalon Studio
- Test Execution: Results of automated test runs

## 🔁 Test Process
- Analyze requirements and identify test scenarios
- Design test cases for core functionalities
- Develop automated test scripts in Katalon
- Execute test scripts and capture results
- Analyze outcomes and report issues

## ✅ Key Highlights
- Automated testing for core features of Pinterest
- Applied black-box testing techniques
- Improved test coverage and execution efficiency

## 🚀 Outcome
- Ensured core functionalities work as expected
- Reduced manual testing effort through automation
- Demonstrated hands-on experience in test automation
